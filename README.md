### Running Instruction
 - install Java 11 [here](https://www.oracle.com/technetwork/java/javase/downloads/index.html)

### Tech Stack

Backend Part uses following libraries and frameworks:
 - Java 15 
 - Spring Boot 2.3.5.RELEASE
 - Maven 3.6.3
 - Lombok  1.18.16
 - Graphql 8.0.0
 - H2 database 1.4.200
 - Google Guava 11.0.2
 - Fasterxml Jackson Core 2.11.3
